
# joelvt

https://joelvt.dev/

My own corner of the internet. For anything and everything, but mainly to build and learn.

## Stack

- Framework: next.js
- ORM: prisma
- Database:
  - postgres (neon)
- Storage: S3
- Emails: nodemailer
- Auth: better auth
- Deployment: Vercel 
- APIs:
  - Movies: TMDb (edge server is in LA)
  - Notifications to self: Discord
- Other bits:
  - Components: shadcn
